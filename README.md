**Para obter a lista de UF:**

    UF.estados
*Retorna:*

    [{ label: 'SP', value: 'SP' }]

____
**Array de objetos com as cidades do estado.**

    UF["AC"]
*Retorna:*

    [{ label:  "Acrelândia", value:  "Acrelândia"  }]
